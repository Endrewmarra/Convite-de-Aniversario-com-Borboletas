# 🎵 Arquivos de Áudio

Este diretório contém os arquivos de áudio para o convite.

## Como adicionar sua música

1. Coloque um arquivo de áudio no formato MP3 neste diretório
2. Renomeie o arquivo para: **`background.mp3`**
3. Salve e pronto! O player usará a música local

### Especificações recomendadas:
- **Formato**: MP3
- **Bitrate**: 128 kbps (balanceado entre qualidade e tamanho)
- **Duração**: 2-5 minutos (será reproduzido em loop)
- **Tamanho**: Até 5MB (para carregamento rápido)

### Fallback
Se nenhum arquivo local for encontrado, o player usará uma música padrão da internet automaticamente.

## Recursos de música gratuita:
- [Bensound](https://www.bensound.com/) - Música livre para uso pessoal
- [Free Music Archive](https://freemusicarchive.org/) - Variedade de gêneros
- [Incompetech](https://incompetech.com/) - Áudio livre royalty-free
